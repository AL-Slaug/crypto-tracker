# 🚀 Crypto Tracker

A responsive cryptocurrency tracking web app built with vanilla JavaScript and Vite.

## ✨ Features

- 📡 Live data fetching from CoinGecko API
- 🔍 Search by cryptocurrency name or symbol
- 🔃 Sort by rank, name, price, 1h%, 24h%, market cap, volume
- 📄 Pagination with dots for large datasets
- 📊 7-day sparkline charts for each coin
- 📱 Fully responsive — mobile, tablet, desktop
- 🍔 Burger menu for mobile navigation
- 🔽 Smart dropdown — opens up or down based on available space
- ⬆️ Scroll to top button
- 🔄 Auto-refresh every 60 seconds

## 🛠️ Technologies

- Vanilla JavaScript (ES6+)
- SCSS (variables, nesting, mixins)
- Vite
- CSS Grid & Flexbox
- CoinGecko API

## 📦 Getting Started
```bash
npm install
npm run dev
```
